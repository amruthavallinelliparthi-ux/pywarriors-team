# Student Record Management System

## Problem Chosen
Problem 1: Student Record Management (CRUD)

## Description
This is a terminal-based Python application developed for PyTech Arena 2026 conducted by JNTU-GV. It performs CRUD operations on student records using OOP and File Handling.

## Features
- Add Student
- View Student
- Update Student
- Delete Student
- Data stored in data.txt

## How to Run
Run the following command in terminal:
python main.py
