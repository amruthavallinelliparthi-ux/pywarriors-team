# Library Management System

## Problem Chosen
Problem 2: Library Management System (CRUD)

## Description
This is a terminal-based Python application developed for PyTech Arena 2026 conducted by JNTU-GV. It manages book records using OOP and File Handling.

## Features
- Add Book
- Issue Book
- Return Book
- Delete Book
- View Books

## How to Run
Run the following command in terminal:
python app.py
